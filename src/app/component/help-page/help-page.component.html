コマ選択

Ctrl + 左クリック
コマの選択と選択解除を切り替えます。選択中のコマの上部には矢印アイコンが表示されます。
範囲選択
テーブル上をドラッグすると、範囲内のオブジェクトをまとめて選択できます。
選択中コマへの付与
バフ、デバフ、テンプレートは現在選択されているコマへ付与されます。
行動完了
ラウンド中はキャラクターのチャットパレット、またはキャラコマの右クリックメニューから行動完了を切り替えられます。名前横には常に行動状態アイコンが表示され、未行動はグレー、完了済みはチェックマークになります。完了時はメインチャットにログが流れます。

パネルとファイル

キャラコマの詳細
コマを右クリックして詳細を開きます。キャラクターシートでは画像、ステータス、チャットパレットなどを編集できます。
キャラコマの効果管理
コマを右クリックし、チャットパレット表示の下にある「効果管理を開く」から、そのキャラを選択した状態で効果管理を開けます。
zip読み込み
保存済みzipはブラウザ画面へドロップして読み込みます。GMモード中にzip形式のキャラを追加した場合、そのキャラはGM作成キャラとして扱われます。
キャラzip保存
キャラコマをzip保存すると、そのキャラに登録されたバフ/デバフテンプレートと付与中効果も一緒に保存されます。読み込み時は新しいキャラへ紐づけ直されます。
外部公開
公開用バッチを起動した場合、cloudflared が表示する https://*.trycloudflare.com/ のURLを共有します。

ステータス参照

書式 説明
2d6+{ステータス名} 送信元キャラのステータスを参照して判定します。対象ステータスにバフ/デバフがある場合は反映されます。

例: 2d6+{筋力} は送信元キャラの筋力を加算します。

リソース操作

書式 説明
:リソース名+値 送信元キャラのリソースを増加します。値にはBCDice、四則演算、{ステータス名} 参照を使用できます。
:リソース名-値 送信元キャラのリソースを減少します。式全体を計算した結果が反映されます。
:リソース名=値 送信元キャラのリソースを指定値にします。
M 接尾辞。最大値を超えないようにします。
! 接尾辞。ダメージ計算が回復に反転するなど、リソースに与える値が0以下になるケースを防ぎます。

例: :HP-1d6 はHPからBCDiceの結果を減らします。

例: :HP-10+5 は -10+5 を計算し、HP-5として扱います。

例: :HP-5+{防護} は防護を参照して最終値を計算します。

例: :HP+30M! は最大値を超えず、増減反転も防ぎます。

ラウンド管理

書式 説明
/round +1 ラウンドを1増加します。GMモード中のユーザーのみ実行できます。メインチャットに 【1ラウンド目】 のように表示され、バフ/デバフの残りラウンドも減少します。
/round reset ラウンドを0に戻し、付与中の効果と行動完了状態を削除します。GMモード中のユーザーのみ実行できます。戦闘終了時のリセットに使用します。

ラウンドが0以外の場合、各チャットには 1R のようなラウンド注釈が付きます。効果管理画面のラウンド表示には、テーブル上コマの行動完了数が 2/3 のように表示されます。ラウンドが1以上の時は、テーブル上の全キャラが行動完了になるまで次ラウンドへ進めません。ラウンド進行時、各キャラの行動完了状態はリセットされます。

バフ/デバフ

書式 説明
/buff 効果名/対象+数値/ラウンド 対象ステータスへ数値を加算する効果を、指定ラウンド分付与します。
/buff 効果名/対象-数値/ラウンド 対象ステータスから数値を減算する効果を、指定ラウンド分付与します。
/buff 効果名/対象*倍率/ラウンド 対象ステータスへ乗算補正を行う効果を付与します。半減などに使用できます。
/buff 効果名/自由記述/ラウンド ステータスを指定せず、説明文のみの効果を付与します。
/buff テンプレート名 送信元キャラに登録済みのテンプレートを、選択中のコマへ付与します。

例: /buff 強化/筋力+2/2 は「強化」という名前の筋力を+2する効果を2ラウンド付与します。

例: /buff 有利/次の判定に有利/1 は自由記述の効果を1ラウンド付与します。

全く同じ効果は同じ対象へ重複付与されません。付与中の効果はキャラコマ上にも表示されます。

効果管理画面

メニューの「効果管理」から、ラウンド、付与中効果、コマごとのテンプレートを管理できます。テンプレートは「ステータス補正」と「自由記述」を切り替えて登録できます。

GMモード

切り替え
GMモードは接続状態やルーム作成者に依存せず、接続情報パネルから自分で切り替えます。
有効化時の確認
有効化時には、秘匿中のステータスやGM向けチャット内容が見えるようになることを確認してからONにします。
GM作成キャラ
GMモード中に作成したキャラ、またはGMモード中にzipから追加したキャラはGM作成キャラになります。非GMは詳細表示やチャットパレット選択ができません。
再接続
GMモードは各ブラウザのローカル設定です。再接続後も必要に応じて接続情報パネルで状態を確認し、切り替えてください。

ステータス秘匿

設定方法
キャラクターシートの ステータス秘匿 を有効にします。
非GMからの見え方
マウスオーバー時の概要パネル、インベントリ上の簡易ステータス表示では、値が ?? になります。
GMからの見え方
GMモードのユーザーには、秘匿中のステータスも通常通り表示されます。
リソース操作コマンド
秘匿中キャラのリソース操作では、非GMには元コマンドの式、参照ステータス、現在値、更新後値、最終合算値が伏せられます。GMには詳細が表示されます。

非GM表示例: :HP ?? / キャラ名 HP: ?? -> ??

GM表示例: :HP-5+{防護} / キャラ名 HP: 20 -> 17 (-5+{防護} = -3)
